Build a production-grade enterprise web application called **Prospect Assist AI** for the IDBI Bank Hackathon.

## Objective

Prospect Assist AI is an AI-powered Lead Intelligence, Customer Relationship Management (CRM), and Loan Origination Support Platform that helps Relationship Managers and Credit Managers identify, analyze and convert high-quality retail lending prospects.

The system should feel like a real internal banking application used by IDBI Bank employees.

This is NOT a public customer portal.

The application should resemble enterprise banking software like Salesforce Financial Services Cloud, SAP Banking, Finacle CRM, Oracle Banking CRM, or Microsoft Dynamics.

--------------------------------------------------

IMPORTANT

This is ONLY a frontend prototype.

There should be

- No Backend
- No Python
- No Express
- No FastAPI
- No Database
- No Firebase
- No Supabase
- No Authentication Server

Everything should work using

/public/data/*.json

and mocked API service functions.

Every API call should have a random delay between 800–1800 ms to simulate real backend processing.

Never mention dummy data anywhere in the application.
